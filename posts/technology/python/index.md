# python总结

## Python 相关问题
### Python ImportError: No module named _bz2
```bash
sudo apt-get install libbz2-dev
#重新编译Python
```
### ImportError: Missing optional dependency 'openpyxl'
```bash
pip install openpyxl
#重新编译Python
```

###  重置linux的python

```bash
unset PYTHONHOME
unset PYTHONPATH
```

###  安装pip
- 下载get-pip.py
```bash
wget https://bootstrap.pypa.io/get-pip.py
```
- 安装pip
```bash
python get-pip.py
```
